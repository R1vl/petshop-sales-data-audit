# Описание данных продаж зоомагазина (2025)

## Контекст
Данные представляют собой агрегированные продажи зоомагазина за 2025 год на уровне товарной позиции.  
Исходные данные содержат исходные коммерческие наименования товаров и брендов, которые **не публикуются**.  

В проекте все колонки были **переименованы в стандартизированный английский формат** для удобства анализа.

Сами исходные данные не включены в репозиторий.

---

## Структура датасета (переименованные колонки)

| Колонка | Тип данных | Описание |
|---------|------------|----------|
| `product_name` | category | Наименование товара |
| `category` | category | Категория товара |
| `qty` | int | Количество проданных единиц |
| `unit` | category | Единица измерения |
| `cost_share_pct` | float | Доля себестоимости в процентах от цены продажи |
| `cost` | float | Себестоимость единицы товара |
| `cost_total` | float | Общая себестоимость продаж товара |
| `cost_total_no_vat` | float | Общая себестоимость без НДС |
| `cost_vat` | float | НДС в себестоимости |
| `markup_pct` | float | Наценка в процентах |
| `markup_total` | float | Сумма наценки |
| `markup_share_pct` | float | Доля наценки от суммы продажи |
| `sales_total_no_discount` | float | Сумма продажи без учета скидок |
| `discount_pct` | float | Размер скидки в процентах |
| `discount_total` | float | Сумма предоставленной скидки |
| `sales_share_pct` | float | Доля продаж товара от общей выручки |
| `margin_pct` | float | Маржинальность в процентах |
| `avg_price` | float | Средняя цена продажи за единицу |
| `sales_total` | float | Сумма продаж с учетом скидок |
| `vat` | float | НДС в сумме продаж |
| `returns_qty` | int | Количество возвращенных единиц |
| `cost_total_with_returns` | float | Себестоимость с учетом возвратов |
| `returns_total` | float | Сумма возвратов |
| `sales_total_with_returns` | float | Сумма продаж с учетом возвратов |
| `markup_total_with_returns` | float | Сумма наценки с учетом возвратов |

---

## Примечания

- Все показатели представлены в агрегированном виде за 2025 год.  
- Пропуски и неконсистентные значения были обработаны на этапе очистки данных (см. ноутбук `01_data_loading_and_cleaning.ipynb`).  
- Некоторые расчетные метрики (маржа, наценка, доля себестоимости) могут иметь некорректные значения в исходных данных; в проекте они проверяются и фиксируются на этапе EDA.

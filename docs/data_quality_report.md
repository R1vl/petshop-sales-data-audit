# Отчёт по качеству данных

## Основные проблемы данных
- Несогласованность расчетных метрик
- Некорректные процентные показатели
- Артефактные экстремальные значения
- Разные единицы измерения в qty

## Масштаб проблемы
- Процентр корректных данных: ~16.5%
- Покрытие выручки: ~2%

## Принятое решение
Анализ ассортимента, скидок и прибыли остановлен,
так как выводы не будут репрезентативны.

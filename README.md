# Аудит качества данных продаж зоомагазина

## Контекст
Проект выполнен на реальных коммерческих данных продаж зоомагазина за 2025 год.
Данные агрегированы на уровне товара за год.

В связи с конфиденциальностью коммерческих данных исходные данные не публикуются.

## Цель проекта
Оценить пригодность данных для анализа выручки, маржинальности,
ассортимента и ценовой политики.

## Роль
Аналитик данных

## Источник данных
Внутренние коммерческие данные компании (агрегированные, без временного разреза).

## Ключевой результат
В ходе EDA и валидации метрик выявлено, что после исключения артефактов
корректными остаются ~16.5% записей, покрывающих ~2% выручки.
Дальнейший анализ признан недостоверным и остановлен.

## Структура проекта
- notebooks/ — код анализа и EDA
- docs/ — аналитические выводы
- data/ — описание структуры данных (без самих данных)

## Используемые инструменты
Python (pandas, matplotlib, seaborn, statsmodels), Jupyter Notebook
